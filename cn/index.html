您好，欢迎来到音乐！

Musicing 是一款轻量、简单的软件，用于构建和拥有您的音乐博客。

音乐根据 GPLv3 许可证发布，按原样提供，我们对其误用不承担任何责任。

第一步，使用密码框和盐字段创建要插入配置文件中的哈希值。请记住手动设置盐值。

当您要使用有限的 Web 服务器或 phpfpm 用户在 PHP 进程上下文中运行 Musicing 时，您必须遵循一些简单的指令以获得最佳的首次设置：
  1. 检查网络应用 Private 路径中“data”文件夹的写入权限；并在配置文件中设置其路径。
  2. 设置默认区域设置。
  3. 设置 FILE_MAX_SIZE（请记住，某些 PHP 设置也可能限制音乐的上传行为）
  4. 设置 BLOG_MAX_POSTS 以限制博客中可见帖子的数量。
  5. 将 PAGINATION 设置为 true 或 false，以允许更多公众访问您的数据。

您可以通过 http://yoursitename.xyz/ 访问您的头像。使用管理视图的密码登录。将所有资源拖放到浏览器窗口中。

歌曲文件命名法：“Playlist~Song Title.mp3”。
