Ciao e benvenuto su Musicing!

Musicing è un software on-premise leggero e semplice per creare e gestire il tuo blog musicale.

La musica è rilasciata sotto licenza GPLv3, viene fornita AS-IS e non ci assumiamo alcuna responsabilità per il suo uso improprio.

Primo passaggio, utilizzare la casella della password e i campi salt per creare l'hash da inserire nel file di configurazione. Ricordati di impostare manualmente lì anche il valore del sale.

Poiché eseguirai Musicing nel contesto del processo PHP, utilizzando un server Web limitato o un utente phpfpm, devi seguire alcune semplici direttive per una prima configurazione ottimale:
  1. Verifica i permessi di scrittura della tua cartella "dati" nel percorso privato della tua web app; e imposta il suo percorso nel file di configurazione.
  2. Imposta la lingua predefinita.
  3. Imposta FILE_MAX_SIZE (ricorda che alcune impostazioni PHP potrebbero limitare anche il comportamento di caricamento di Musicing)
  4. Imposta BLOG_MAX_POSTS per limitare il numero di post visibili nel blog.
  5. Imposta PAGINATION su true o false per garantire un maggiore accesso pubblico ai tuoi dati.

Puoi accedere al tuo avatar da http://yourservername.xyz/<your_avatar>. Accedi con la password per la vista amministratore. Trascina tutte le tue risorse nella finestra del browser.

Nomenclatura del file del brano: "Playlist~Titolo del brano.mp3".
